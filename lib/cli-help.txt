Usage
$ markserv <file/dir>
$ readme (serve closest README.md file)

Options
	--port, -p,  HTTP port [port] (80)
	--livereloadport, -l  LiveReload port [port/false] (35729)
	--autoreload  Reload mode: manual (default), auto, false
	--browser, -b  Launch browser (true)
	--silent, -i  Silent (false)
	--address, -a  Serve on ip/address [address] (localhost); comma-separated for multiple (e.g. "192.168.1.1,localhost")
	               Env: MARKSERV_ADDRESS (avoids storing address in shell history)
	--verbose, -v  Verbose output (false)
	--show-all-dir  Show all directories even if they contain no markdown (false)
	--force-replace  Kill existing markserv on the same port and restart (false)
	--poll  Use polling instead of native fs watchers (true)
